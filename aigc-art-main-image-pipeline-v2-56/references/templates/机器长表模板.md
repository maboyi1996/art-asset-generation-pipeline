# 机器长表模板

## 文件名

`main-image-production-table.md`

## 必须包含

1. 项目概览。
2. 角色身份锚点表。
3. 人物候选去向审计表。
4. 污染标签清洗记录表。
5. 全局生产任务表。
6. 分集使用映射表。
7. 生产审计摘要。

## 人物候选去向审计表字段

| 字段 |
|---|
| `candidate_id` |
| `raw_name` |
| `normalized_name` |
| `presence_type` |
| `scene_ids` |
| `candidate_scope` |
| `base_asset_id` |
| `priority_level` |
| `same_asset_evidence` |
| `final_disposition` |
| `task_id` |
| `alias_to` |
| `grouped_into` |
| `excluded_reason` |
| `disposition_evidence` |
| `disposition_uncertainty_note` |

本表是人物候选闭环审计章节，不是全局生产任务表。不得把本表字段加入全局生产任务表字段。

本表只能说明人物候选最终去向。B/C 可生产人物不得只出现在本表中；其 `production_task`、`state_task` 或 `template_task` 去向必须能在 `全局生产任务表` 找到对应 `task_id`。

污染标签清洗记录不得写入本表。`raw_name` 和 `normalized_name` 必须是已进入 `asset-candidate-list` 的真实人物候选，不得包含动作、情绪、语气、声音、瞬时表情、组合词或未消歧可转移称谓。

## 污染标签清洗记录表字段

| 字段 |
|---|
| `cleanup_id` |
| `raw_label` |
| `normalized_subject` |
| `cleanup_type` |
| `source_scene_ids` |
| `source_evidence` |
| `handling_result` |
| `target_candidate_id` |
| `review_note` |

本表只记录污染标签清洗过程，不是人物候选去向审计表，也不是全局生产任务表。

本表不得参与人物候选数、S/A/B/C 数、生产任务数、`ready_task_count`、`manual_review_task_count` 或 `excluded_count` 统计。`raw_label` 是机器长表中除原文证据外唯一允许保留污染标签的位置。

## 全局生产任务表字段

| 字段 |
|---|
| `task_id` |
| `asset_type` |
| `asset_name` |
| `state_name` |
| `priority_level` |
| `same_asset_evidence` |
| `task_type` |
| `anchor_task_id` |
| `review_status` |
| `review_reason` |
| `episode_id` |
| `source_evidence` |
| `prompt` |
| `edit_instruction` |
| `negative_prompt` |
| `review_focus` |

全局生产任务表字段只使用上表列举字段；新增字段必须先更新 `任务类型字段审核规则.md`。人物候选去向审计表字段和污染标签清洗记录表字段不属于全局生产任务表字段。

全局生产任务表是导入中台的一键生成 source table。所有来自 `global-task-draft` 的生产任务行必须进入本表；人物候选去向审计表、污染标签清洗记录表、角色身份锚点表和分集使用映射表都不得替代本表。

`text_to_image` 任务必须填写 `prompt` 和 `negative_prompt`。`image_text_edit` 任务必须填写 `edit_instruction`、`negative_prompt` 和有效 `anchor_task_id`。

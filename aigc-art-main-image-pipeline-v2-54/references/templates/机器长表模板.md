# 机器长表模板

## 文件名

`main-image-production-table.md`

## 必须包含

1. 项目概览。
2. 角色身份锚点表。
3. 人物候选去向审计表。
4. 全局生产任务表。
5. 分集使用映射表。
6. 生产审计摘要。

## 人物候选去向审计表字段

| 字段 |
|---|
| `candidate_id` |
| `raw_name` |
| `normalized_name` |
| `presence_type` |
| `scene_ids` |
| `candidate_scope` |
| `base_asset_id` |
| `priority_level` |
| `same_asset_evidence` |
| `final_disposition` |
| `task_id` |
| `alias_to` |
| `grouped_into` |
| `excluded_reason` |
| `disposition_evidence` |
| `disposition_uncertainty_note` |

本表是人物候选闭环审计章节，不是全局生产任务表。不得把本表字段加入全局生产任务表字段。

本表只能说明人物候选最终去向。B/C 可生产人物不得只出现在本表中；其 `production_task`、`state_task` 或 `template_task` 去向必须能在 `全局生产任务表` 找到对应 `task_id`。

## 全局生产任务表字段

| 字段 |
|---|
| `task_id` |
| `asset_type` |
| `asset_name` |
| `state_name` |
| `priority_level` |
| `same_asset_evidence` |
| `task_type` |
| `anchor_task_id` |
| `review_status` |
| `review_reason` |
| `episode_id` |
| `source_evidence` |
| `prompt` |
| `edit_instruction` |
| `negative_prompt` |
| `review_focus` |

全局生产任务表字段只使用上表列举字段；新增字段必须先更新 `任务类型字段审核规则.md`。人物候选去向审计表字段不属于全局生产任务表字段。

全局生产任务表是导入中台的一键生成 source table。所有来自 `global-task-draft` 的生产任务行必须进入本表；人物候选去向审计表、角色身份锚点表和分集使用映射表都不得替代本表。

`text_to_image` 任务必须填写 `prompt` 和 `negative_prompt`。`image_text_edit` 任务必须填写 `edit_instruction`、`negative_prompt` 和有效 `anchor_task_id`。

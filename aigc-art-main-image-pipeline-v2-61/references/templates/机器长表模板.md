# 机器长表模板

## 文件名

`main-image-production-table.md`

## 必须包含

1. 项目概览。
2. 人物评级前证据账本。
3. 场景评级前证据账本。
4. 道具评级前证据账本。
5. 全局资产注册表。
6. 别名归并表。
7. 状态索引表。
8. 状态线索去向表。
9. 角色身份锚点表。
10. 人物候选去向审计表。
11. 泛称/群体模板覆盖表。
12. 污染标签清洗记录表。
13. 全局生产任务表。
14. 分集使用映射表。
15. 生产审计摘要。

机器长表是唯一完整 source-of-truth。所有评级前证据账本、生产任务、状态、提示词、编辑指令、审核状态、证据、分集引用、候选去向、污染清洗和审计结果都必须在本文件中可追踪。

本表是人物候选闭环审计章节，不是全局生产任务表。不得把本表字段加入全局生产任务表字段。

B/C 可生产人物不得只出现在本表中；其 `production_task`、`state_task` 或 `template_task` 去向必须能在 `全局生产任务表` 找到对应 `task_id`。

污染标签清洗记录不得写入本表。`raw_name` 和 `normalized_name` 必须是已进入 `asset-candidate-list` 的真实人物候选，不得包含动作、情绪、语气、声音、瞬时表情、组合词或未消歧可转移称谓。

|---|

| `asset_type` |

| `priority_level` |

| `review_reason` |
| `episode_id` |
| `source_evidence` |
| `prompt` |
| `edit_instruction` |
| `negative_prompt` |
| `review_focus` |

全局生产任务表字段只使用上表列举字段；新增字段必须先更新 `任务类型字段审核规则.md`。人物候选去向审计表、泛称/群体模板覆盖表、污染标签清洗记录表、状态线索去向表的字段不属于全局生产任务表字段。

全局生产任务表是导入中台的一键生成 source table。所有来自 `global-task-draft` 的生产任务行必须进入本表；人物候选去向审计表、污染标签清洗记录表、角色身份锚点表和分集使用映射表都不得替代本表。

`text_to_image` 任务必须填写 `prompt` 和 `negative_prompt`。`image_text_edit` 任务必须填写 `edit_instruction`、`negative_prompt` 和有效 `anchor_task_id`。

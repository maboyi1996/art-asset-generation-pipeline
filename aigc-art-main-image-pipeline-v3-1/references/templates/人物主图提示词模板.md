# 人物主图提示词模板

## 用途

供 `scripts/render_core_prompts.py` 根据 `production-source/core-prompt-slots.md` 渲染人物主图中文 core prompt。V3.1 中 `prompt-generation.md` 只填写槽位，不直接手写整段 prompt。

本模板不包含题材世界段、画法段或风格前缀。中台会在本 core prompt 前拼接运营选择的风格前缀。

## 核心原则

本模板原始设计有 A「题材世界」和 B「人物画法」两层；V3.1 改为由中台后续拼接，所以模板正文只生成 C 之后的 core prompt。D/E 只写角色资产事实、可见形象和表演状态，不写题材词、画法词、渲染词或风格前缀。C 段构图/限制规则和末尾负向约束是唯一防错位置；D/E 段不得写“不要、不能、避免、不得、无XX设定、不加入XX”等负向或排除式句子。未在项目或剧本中存在的元素直接省略，不声明其不存在。负向约束只负责生产防错，不承担题材或画法控制。

## 输入字段

- `identity`
- `face_features`
- `hair_and_decoration`
- `body_posture`
- `costume_visible_decoration`

## 主图 core prompt 模板

```text
图像类型：
人物胸像主设定图。

画幅比例：
竖版 9:16。

构图规格：
单人胸像，只有一个人物主体，干净中性灰背景，人物居中展示。画面必须清晰看见脸部、颈部、肩部和上胸，人物轮廓明确。

背景规则：
#808080 色号代表的干净中性灰底图。

角色身份：
[姓名 / 年龄 / 性别 / 身份 / 阶层 / 职业 / 一句话定位]

脸部特征：
[脸型 / 骨相 / 五官 / 肤色 / 皮肤状态 / 面部识别细节]

发型与修饰：
[发型 / 胡茬 / 妆造 / 面部小物件 / 疤痕 / 标记]

身形与体态：
[体型 / 肩颈 / 姿态基础 / 年龄感]

服装与可见装饰：
[服装轮廓 / 材质 / 新旧状态 / 局部装饰的名称、位置、材质、颜色和可见状态]

配色与识别重点：
[主色 / 辅助色 / 角色一眼不能丢的视觉记忆点]

眼神：
[视线方向 / 眼神状态 / 心理暗示]

表情：
[表情强度 / 嘴角 / 情绪克制程度]

身体语言：
[头部角度 / 肩颈状态 / 姿态张力]

情绪核心：
[这个角色此刻最重要的气质，不写剧情动作]

负向约束：
不要环境，不要其他人物，不要剧情截图，不要动作姿势，不要复杂剧情背景；不要错误年龄、性别、身份或角色类型；不要让手部、武器、道具、服装或装饰遮挡或抢过脸；不要水印，不要可读文字。
```

## 约束

- 最终提示词必须是中文 core prompt。
- 不写题材世界段。
- 不写画法段。
- 不写 `{{include:...}}`。
- 不把风格库前缀写入本模板。
- D/E 段只写正向可见事实，不写“不要、不能、避免、不得、无XX设定、不加入XX”等负向或排除式句子。
- 没有设定的元素直接省略，不写不存在元素的否定声明。
- 防遮脸、防多人物、防场景、防动作等生产防错只写在负向约束或 `negative_prompt`。
- 不把剧情事件替代为可见外貌描述。
- 人物主图必须是单人，中性背景，不写关系戏、动作戏或场景戏。

# 人审 DOCX 表格模板

## 文件名

`main-image-review-table.docx`

## 作用

供 `review-docx-generation.md` 通过 `artifact-manifest.json` 从 `production-source/main-image-production-table.md` 派生人审 DOCX。DOCX 只服务人工检查和贴图，不是生产 source-of-truth。

## 生成顺序

按集输出，每集固定包含：

1. `第X集`
2. `人物表`
3. `场景表`
4. `道具表`

## 表格列

### 人物表

| 人物 | 本集状态与外貌 | 评级 | 人设图 | 5视图 |
|---|---|---|---|---|

### 场景表

| 场景 | 描述 | 评级 |  |  |
|---|---|---|---|---|

### 道具表

| 道具 | 描述与归属 | 评级 |  |
|---|---|---|---|

## 字段来源

- `人物`、`场景`、`道具`：来自生产源表的 `asset_name`。
- `本集状态与外貌`、`描述`、`描述与归属`：来自生产源表中对应任务和分集映射的 `state_name`、`source_evidence`、`review_focus` 等非提示词字段。
- `评级`：来自生产源表的 `priority_level`，展示为 `S级/A级/B级/C级`。
- 图片列：只输出空白占位，不读取、不复制、不生成图片。

## 禁止内容

- 不输出 `基础提示词` 列。
- 不输出 `prompt`、`edit_instruction`、`negative_prompt`。
- 不输出完整 core prompt。
- 不输出风格前缀。
- 不输出 evidence ledger、污染清洗记录、人物候选去向台账、状态索引完整表或完整生产任务长表。
- 不从 `audit/` 补字段。
- 不反向改变生产源表中的任务、状态、评级、审核状态、提示词、JSON 字段或分集映射。

## 空表口径

某集没有某类资产时，保留该类小标题，并输出一行：

| 名称列 | 描述列 | 评级列 | 图片占位列 |
|---|---|---|---|
| 本集无新增/使用项 |  |  |  |

人物表按 5 列输出空行；场景表按 5 列输出空行；道具表按 4 列输出空行。

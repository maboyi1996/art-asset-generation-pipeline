# 角色状态编辑模板

## 用途

供 `scripts/render_core_prompts.py` 根据 `production-source/core-prompt-slots.md` 渲染人物状态图中文 `image_text_edit` core instruction。V3.1.1 中 `prompt-generation.md` 只填写槽位，不直接手写整段编辑指令。

本模板不包含题材世界段、画法段或风格前缀。中台会在本编辑指令前拼接运营选择的风格前缀，或按中台编辑任务规则使用本指令。

## 核心原则

本模板只生成同一人物的状态编辑 core instruction。题材世界和人物画法由中台后续拼接，本模板不得写题材词、画法词、渲染词或风格前缀。状态差异只写可见外观变化；负向约束只负责防止换脸、改年龄、改比例、加道具、加场景、加人物、加文字或水印。

## 输入字段

- `identity_reference_image`
- `identity_preservation`
- `state_delta_prompt`
- `composition_preservation`

## 模板

```text
任务类型：
image_text_edit。

输入图：
[identity_reference_image]

身份保持规则：
以已确认的身份锚点图为同一人物基准。必须保持同一个人的脸型、骨相、五官比例、眼距、眼型、鼻子、嘴型、下颌、肤色倾向、年龄段、体型和核心眼神气质。

只允许改变以下可见状态细节：
[state_delta_prompt]

构图保持：
保持单人胸像或半身人物主设定图，使用干净中性背景。不要加入道具、场景、其他人物或动作戏。

负向约束：
不要变成另一个人；不要改变脸型、骨相、五官比例、眼距、眼型、鼻子、嘴型、下颌、肤色倾向或年龄段；不要美化重设计；不要年龄错误；不要通用替代脸；不要加入道具、房间、地点、其他人物、可读文字或水印。

画幅比例：
竖版 9:16。
```

## 约束

- 只能用于已有 `anchor_task_id` 和 `identity_reference_image` 的状态图。
- 状态差异只写可见外观变化。
- 不把道具、地点、动作、关系或情绪事件写成状态差异。
- 不写题材世界段、画法段、`{{include:...}}` 或风格库前缀。

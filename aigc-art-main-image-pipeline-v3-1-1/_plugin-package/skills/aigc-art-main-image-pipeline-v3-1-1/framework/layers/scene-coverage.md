# scene-coverage.md

## 作用

逐场记录客观出现事实：人物、地点、道具、可见状态线索和原文证据，并区分人物可见出场、VO/提及、纪念物指代等出现类型。

## 输入

`audit/script-intake-result.md`

## 调用文件

`references/rules/场次覆盖记录规则.md`

## 输出结果

`audit/scene-coverage-table.md`

## 边界

只记录场次事实，不提前合并资产、不做候选准入、不评级、不生成任务。污染短语、动作、情绪、语气、组合词可以留在原文证据或覆盖备注中，但不得作为结构化人物候选决策输出。

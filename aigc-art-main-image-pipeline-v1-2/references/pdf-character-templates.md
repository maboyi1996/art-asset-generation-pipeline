```text

This is the identity anchor portrait for all later states of this character. The image must establish the stable casting identity: [face shape], [bone structure], [eye shape and spacing], [nose structure], [mouth shape], [skin undertone], [age band], [gaze quality], [class/status impression], and [core emotional presence].

Wardrobe is secondary in this identity anchor unless it is a permanent identity costume. Use [simple identity-supporting wardrobe] only to support social class, occupation, or story role. Do not let wardrobe replace face identity.

realistic face proportions, distinctive but natural human face, age-appropriate, subtle facial asymmetry, realistic pores, natural skin color variation, believable hairline, emotionally present eyes, not generic attractive person, not AI beauty face, not fashion model, not influencer beauty, not commercial campaign polish.
```

## Character State Edit Prompt Template

Use this for later states of a key character after the identity anchor image is approved.

```text
Generation mode: image_edit_from_identity_anchor
Input image: [identity_reference_image]

Use the provided approved identity reference image as the source of truth for the character's face and casting identity. Preserve the same person: same face shape, bone structure, facial proportions, eye spacing, eye shape, nose structure, mouth shape, jawline, skin undertone, age band, and core gaze quality.

Edit only the visible state details for this episode/state:
[state_delta_prompt: wardrobe / grooming / hair condition / makeup / injury / illness / fatigue / dirt / blood / wetness / class styling, based on script evidence].

Keep a single-character bust portrait or half-body portrait on a plain clean neutral background. No props, no scene, no other characters, no location, no action-scene staging.

Identity preservation:
do not redesign the face, do not make a new person, do not beautify, do not de-age, do not change facial proportions, do not change ethnicity or regional features, do not change body type unless the state evidence requires it, do not change the character's core emotional presence.

State realism:
the new clothing, grooming, injury, illness, fatigue, dirt, blood, wetness, or makeup must look physically real and story-supported, with realistic fabric behavior, skin texture, hair detail, and natural camera limitations.

Negative prompt:
```

```text

Input image: [identity_reference_image]

Start from the approved identity reference image. Preserve recognizable continuity with the same character unless the script explicitly requires an unrecognizable transformation.

Allowed identity-sensitive changes:
[exact changes allowed by evidence, such as older age appearance / visible illness / injury / disfigurement / pregnancy / severe fatigue].

Protected identity traits:
[face shape / eye spacing / nose / mouth / skin undertone / gaze quality / class impression that should remain continuous].

Do not add unrelated redesign. Do not create a new actor. Mark for manual review if the transformation makes identity continuity uncertain.

Negative prompt:
unrelated new person, different actor, beautified redesign, de-aged face unless required, unsupported injury, unsupported illness, exaggerated gore, generic old-age filter, plastic prosthetic look, CG render face, props, scene, other characters, readable text, watermark.
```

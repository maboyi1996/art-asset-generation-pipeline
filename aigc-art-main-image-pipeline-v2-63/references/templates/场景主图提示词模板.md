# 场景主图提示词模板

## 用途

供 `prompt-generation.md` 为场景资产生成主图提示词。

## 输入字段

- `asset_name`
- `state_name`
- `priority_level`
- `source_evidence`
- `space_identity`
- `layout_notes`
- `material_notes`
- `lighting_notes`

## 模板

```text
[Scene Name], [scene type] inside [larger location / institution / household] at [script-supported time or lighting condition]. Main scene reference image, no recognizable people unless the scene asset itself requires a crowd or event state.

Spatial identity: [accurate room / street / hall / courtyard / institution identity], not [common wrong location].
Layout: [entrance], [main space], [visual center], [background structure], [movement path if needed].
Core story elements: [required fixed set pieces / furniture / symbols / traces].
Materials: [wall / floor / furniture / fabric / metal / wood / stone], realistic surface behavior, believable scale, restrained texture density.
Lighting: [script-supported source], realistic falloff, limited camera dynamic range, natural dark areas, no decorative CG lighting.
Mood: [story-supported atmosphere], practical real-location or real-set photographic reference.

Photographic realism: real photograph, natural lens softness, imperfect framing, low-information large surfaces, physically plausible light and materials, no game CG, no concept art, no PBR asset look.

Avoid: wrong location, wrong class, wrong era, unrelated clutter, recognizable people unless required, poster composition, HDR clarity, over-sharpening, procedural texture noise, mirror-like CG floors, UI artifacts, watermark, readable modern text.
```

## 约束

- 场景主图默认空场或近空场。
- 不为普通场景自动添加人群。
- 时间、光照和空间状态必须来自剧本证据或保守推断。

# 输出质量守卫

## 目的

检查最终交付是否满足 V2.63 输出契约。

本守卫只检查文件、章节、字段、引用和审计状态，不重新抽取资产、不重新拆状态、不重新评级、不补任务、不改变输出。

## 必查项

- 是否输出 `main-image-production-table.md`。
- 是否输出 `prompt-manifest.json`。
- 是否输出 `main-image-review-table.md`。
- 机器长表是否包含项目概览、人物评级前证据账本、场景评级前证据账本、道具评级前证据账本、全局资产注册表、别名归并表、状态索引表、状态线索去向表、角色身份锚点表、人物候选去向审计表、泛称/群体模板覆盖表、污染标签清洗记录表、全局生产任务表、分集使用映射表和生产审计摘要。
- 人审短表是否只包含项目概览、角色身份锚点摘要、分集使用映射摘要和 blocking 摘要。
- 人审短表是否没有放入三类完整证据账本、完整全局资产注册表、状态索引表、人物候选去向台账、泛称/群体模板覆盖表、污染标签清洗记录表或完整生产任务长表。
- 机器长表中的全局生产任务表字段是否只使用 `任务类型字段审核规则.md` 和 `机器长表模板.md` 允许的生产字段。
- 全局生产任务表是否包含 `base_asset_id` 和 `state_asset_id`；`base_asset_id` 必须能回指全局资产注册表，状态任务的 `state_asset_id` 必须能回指状态索引表。
- 全局资产注册表是否不包含旧版状态资产列表字段或任何状态 ID 列表；状态只能出现在状态索引表。
- 状态索引表和状态线索去向表是否只包含 S/A 状态范围内的数据；B/C、泛称、群体、普通背景资产或无法指向 S/A 资产的状态线索不得出现。
- 人物候选去向审计字段、泛称/群体模板覆盖字段、污染标签清洗记录字段、状态线索去向字段是否没有污染全局生产任务表字段。
- `production-audit-result` 是否仍有 `severity: blocking`；如有，最终交付必须标记未通过，不得在终检层修复。
- 每个 `text_to_image` 生产任务是否填写 `prompt` 和 `negative_prompt`。
- 每个 `image_text_edit` 生产任务是否填写 `edit_instruction`、`negative_prompt` 和有效 `anchor_task_id`。
- 每条分集映射是否引用真实 `task_id`，或通过 `disposition_ref` 指向可见排除、非视觉去向或泛称模板覆盖记录。
- `task_type` 是否只使用 `text_to_image` 和 `image_text_edit`。
- `review_status` 是否只使用 `ready`、`manual_review_required`、`excluded`。
- `priority_level` 是否只使用 `S/A/B/C`，状态是否继承 base asset 等级，且状态行只限 S/A。
- B/C 可生产资产是否只生成基础任务或合法排除/模板去向；不得存在 B/C 状态任务、B/C `image_text_edit` 状态链路或 B/C 状态人工复核记录。
- 污染标签是否只出现在三类 evidence ledger 的 raw 字段、`source_evidence` 原文证据或 `污染标签清洗记录表.raw_label`。
- `polluted-label-cleanup-ledger` 是否没有被计入人物候选数、S/A/B/C 数、生产任务数或审核状态统计。
- `prompt-manifest.json` 是否由机器长表投影生成，且没有反向减少、合并、改名、补建或重算机器长表任务。
- 人审短表是否没有反向减少、合并、改名、补建或重算机器长表任务。

## 非阻断项

- 不因项目集数长、B/C 总量低、模板数量少或等级比例异常直接判定最终质检失败。
- 不因 B/C 状态线索没有出现在状态索引表、状态线索去向表或状态人工复核中判定最终质检失败。
- 上述数量异常只能在生产审计或人审摘要中作为 `warning`/`info` 呈现，不能替代逐事实、逐候选、逐状态、逐任务、逐映射的闭环检查。

## 失败处理

发现文件缺失、章节缺失、字段非法、引用断链、提示词字段缺失、污染标签进入结构化名称、机器长表/人审短表/JSON 分工错误或 `production-audit-result` 仍有 blocking 时，返回对应拥有层修复，不在最终质检层重写结果。

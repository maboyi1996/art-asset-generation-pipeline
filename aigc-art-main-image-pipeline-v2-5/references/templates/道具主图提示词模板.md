# 道具主图提示词模板

## 用途

供 `prompt-generation.md` 为道具资产生成主图提示词。

## 输入字段

- `asset_name`
- `state_name`
- `priority_level`
- `source_evidence`
- `object_identity`
- `material_notes`
- `condition_notes`

## 模板

```text
[Prop Name], isolated prop main reference image, clean white background, single object only, no environment, no people, no hands, no readable text, no visible brand logo.

Object identity: [accurate object type], not [common wrong type].
Story state: [condition / owner trace / damage / seal / opened state / blood / wear if supported].
Structure: [core structure 1], [core structure 2], [must-visible component].
Materials: [material 1], [material 2], realistic surface detail, believable scale, correct material thickness, physically plausible reflection behavior.
Condition: [new / used / damaged / stained / sealed / opened], matching script evidence.
Composition: centered product-reference composition, full object visible, enough padding, clean natural shadow.
Lighting: soft neutral studio lighting, controlled highlights, material texture clearly visible.

Avoid: hands, people, extra objects, scene environment, readable text, watermark, wrong object type, wrong material, wrong condition, dramatic scene lighting, PBR product render, luxury catalog polish, fake CGI look.
```

## 约束

- 道具图默认独立白底单物。
- 不自动加入人物、手、环境或品牌文字。

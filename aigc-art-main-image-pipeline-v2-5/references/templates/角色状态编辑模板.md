# 角色状态编辑模板

## 用途

供 `prompt-generation.md` 为人物状态图生成 `image_text_edit` 编辑指令。

## 输入字段

- `task_id`
- `asset_name`
- `state_name`
- `anchor_task_id`
- `identity_reference_image`
- `stable_face_identity`
- `state_delta_prompt`
- `source_evidence`

## 模板

```text
Task type: image_text_edit
Input image: [identity_reference_image]

Use the approved identity reference image as the source of truth for the same person.
Preserve the same face shape, bone structure, facial proportions, eye spacing, eye shape, nose, mouth, jawline, skin undertone, age band, body type, and core gaze quality.

Change only these visible state details:
[state_delta_prompt]

Keep a single-character portrait or half-body portrait on a plain clean neutral background. No props, no scene, no other characters, no action staging.

Negative prompt: different person, changed face shape, changed bone structure, changed eye spacing, changed eyes, changed nose, changed mouth, changed jawline, changed skin undertone, changed age band, beautified redesign, de-aged face, generic replacement face, AI beauty face, plastic skin, waxy skin, over-retouched glamour, props, scene, room, location, other characters, readable text, watermark.

Image size: 9:16
```

## 约束

- 只能用于已有 `anchor_task_id` 和 `identity_reference_image` 的状态图。
- 状态差异只写可见外观变化。
- 不把道具、地点、动作、关系或情绪事件写成状态差异。
